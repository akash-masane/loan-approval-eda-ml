# loan-approval-Eda-Ml
This project focuses on Exploratory Data Analysis (EDA) and basic Machine Learning
to predict loan approval status based on applicant data.

## Objectives
- Understand the dataset through EDA
- Handle missing values and outliers
- Perform feature engineering
- Build and evaluate a classification model

## Steps Performed
- Data cleaning and preprocessing
- Univariate and bivariate analysis
- Encoding categorical variables
- Model training and evaluation
- Feature importance analysis

## Tools & Libraries
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- Jupyter Notebook

## Note
This project was completed as part of structured learning and hands-on practice
to strengthen concepts of data analysis and machine learning.
